In Hindsight

After reaching the deadline of this project the main problem I still faced was the inability to properly measure the Fan rpm using interrupts. In hindsight if I where to create this project from scratch I would make my own interrupt functions using the proper reggisters in a more bit-oriented approach. That principal was implemented on the Tacho Meter project.
