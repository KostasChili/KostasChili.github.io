Tacho Meter Logger

This project aimed in the creation of a tacho meter logger based on Arduino microcontrollers with the use of C++ language in a bit oriented approach.

In short detail we needed a Tacho-Meter with logging capabilities to measure the rpm of various generators over time used in small hydro-electric stations. The meter was composed by :

  • Arduino Nano as the main proccessing unit
  • IR sensor for picking up pulses used to calculate rpm
  • RTC IC for time keeping
  • SD card module for logging
  • LCD screen for data display and buttons for navigation and settings
  • Software
